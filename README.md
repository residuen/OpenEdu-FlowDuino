OpenEdu-FlowDuino
=================

<u>Ein einfaches Programm zum Erlernen der Grundlagen der Regelungstechnik</u><br/>

<b>FlowDuino</b> ist ein einfaches Layout-Programm mit dem Ziel,
die Grundbegriffe der Regelungstechnik interaktiv zu vermitteln.<br/>
Hierbei wird eine Arduino mit Ethernet-Shield über ein grafisches Frontend programmiert. Regelgrößen k&ouml;nnen grafisch dargestellt werden.<br />

<b>Programmiersprache</b><br/>
MyLayout ist in der Programmiersprache Java (Version 7) geschrieben. Ziel sind lauff&auml;hige
Versionen f&uuml;r Windows- und Linux-Systeme.<br/>

<b>Grafikausgabe</b><br/>
F&uuml;r die 2D-Grafikausgabe wird Java2D verwendet.<br/>
