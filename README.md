OpenEdu-MyLayout
================

Ein einfaches Programm für Platinen-Layouts<br/>
Lizenz: GPL 3.0<br/><br/>

MyLayout ist/wird ein einfaches Layout-Programm mit dem Ziel,
in kurzer Zeit Leiterplatten für Nass&auml;tzverfahren herzustellen.<br/><br/>

Sprache.<br/>
MyLayout ist in der Programmiersprache Java (Version 7) geschrieben. Ziel sind lauff&auml;hige
Versionen f&uuml;r Windows- und Lionux-Systeme.<br/><br/>
